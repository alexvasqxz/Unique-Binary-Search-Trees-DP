# Unique-Binary-Search-Trees-DP

<h3>
This program gives the number of unique Binary Search Trees that can be formed given
n Nodes using Dynamic Programming
</h3>
	
<h4>
	Example: 
	<br>
Input : 4
	<br>
Output: 14
</h4>

<h4>
Explanation:
<br>
Given n = 4, there are a total of 14 unique BST's
</h4>
